# 🐯 그로스 마케팅 부트캠프 포트폴리오

### _Data-Driven Growth Marketing Learning Journey_

> **고객과 비즈니스의 문제를 데이터로 발견하고, 분석 결과를 실행 가능한 마케팅 전략으로 연결하는 과정을 기록합니다.**

멋쟁이사자처럼 그로스 마케팅 부트캠프에서 배우고 실습한 내용을 주차별로 정리하는 포트폴리오입니다.
단순한 수업 필기보다 **문제 정의 → 데이터 분석 → 인사이트 도출 → 마케팅 액션 제안**의 흐름을 보여주는 데 집중합니다.

### 🎯 Career Goal

데이터를 단순히 조회하는 데서 멈추지 않고, 다음 질문에 답할 수 있는 그로스 마케터로 성장하는 것이 목표입니다.

- 어떤 고객 문제가 성장을 방해하고 있는가?
- 어떤 지표를 기준으로 문제를 확인해야 하는가?
- 분석 결과는 어떤 마케팅 액션으로 이어져야 하는가?
- 실행한 전략의 성과를 어떻게 측정하고 개선할 것인가?
